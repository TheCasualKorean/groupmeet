Making Team Scheduling Easier!

GroupMeet is the place that encourages easy viewing of everyone's schedule!

Sign Up Sign In

Features of GroupMeet:

  • Each group has a discussion forum that includes a timestammp as well as a history of comments
  • Each group will give the ability to request meetings for 2 or more members within the group
  • Replying to the meeting will allow you to choose between attending, telecommunicate, or not attending
  • Collaborations within the group are for invite only
  • There will be a project planner that includes a board listing for: To Do, In-Progress, Completed, and Need Help
  • Ability for a schedule tracker that can be modified by the user
  • Each schedule is color-coded for easy recognition
  • Schedules that are created have the ability to be shared across platforms
  • Each group page will have a calendar for the week, a tab for any URLs posted as well as comments
  • Any created meeting will show up for all group members
  • Meetings have the ability to be scheduled at the same time for subgroups
